# GarciaFranco_Portfolio

A full-stack MERN portfolio project.

## Features

- Black minimalist theme
- Responsive design
- Clean, modular code

## Tech Stack

- **Frontend**: React (Vite), Framer Motion
- **Backend**: Express.js, Node.js
- **Database**: MongoDB (Mongoose)
- **Styling**: CSS with variables

## Setup

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   cd server && npm install
   cd ../client && npm install
   ```
3. Place your profile image `scifi.png` in `client/src/assets/`
4. Start MongoDB locally
5. Run the project:
   ```bash
   npm run dev
   ```
   This will start both server (port 5000) and client (Vite dev server)

## Project Structure

- `server/` - Express backend
- `client/` - React frontend
- `client/src/assets/` - Static assets (add scifi.png here)
